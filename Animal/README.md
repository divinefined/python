# Tamagotchi

Виртуальная игра-симуляция по уходу за питомцем. Питайте и развлекайте вашего питомца, чтобы сохранить ему жизнь!

## Описание

**Tamagotchi** — это интерактивная игра, где вы ухаживаете за виртуальным питомцем. Питомец нуждается в еде и развлечениях, а его благополучие зависит от ваших действий.

Игра использует многопоточность для реалистичной симуляции жизнедеятельности питомца.

## Требования

- Python 3.6+
- Встроенные модули: `time`, `threading`

## Установка

Не требует установки дополнительных пакетов.

## Использование

```bash
python animal.py
```

После запуска введите команды:
- `feed` — накормить питомца (требует ввести количество еды)
- `play` — поиграть с питомцем (требует ввести количество времени)

## Механика игры

### Параметры питомца

- **Сытость (hunger)** — начинается с 100, уменьшается естественным путем
- **Радость (happiness)** — начинается с 100, влияет на жизнь питомца

### Действия

- **Кормление**: Увеличивает сытость (максимум 100)
- **Игра**: Увеличивает счастье, но уменьшает сытость
- **Проверка статуса**: Питомец умирает, если сытость или счастье достигнут 0

### Цикл жизни

- Каждые 10 секунд сытость и счастье уменьшаются на 10
- Питомец проверяется на жизнеспособность после каждого действия
- Если питомец умер, игра завершается

## Структура кода

### Класс `Tamagotchi`

**Атрибуты:**
- `name` — имя питомца
- `hunger` — уровень сытости (0-100)
- `happiness` — уровень счастья (0-100)
- `is_alive` — жив ли питомец

**Методы:**
- `feed(food)` — накормить питомца
- `play(fun)` — поиграть с питомцем
- `check_status()` — проверить, жив ли питомец
- `live()` — основной цикл жизни (запускается в отдельном потоке)

### Функция `main()`

Управляет игровым процессом, читает команды пользователя и обновляет состояние питомца.
