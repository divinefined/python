# Custom List - My_list

Кастомная реализация списка с дополнительными операциями вычитания и деления.

## Описание

**My_list** — это класс, который наследует встроенный тип `list` и добавляет поддержку операторов вычитания (`-`) и деления (`/`) для работы со списками.

Это демонстрирует мощь полиморфизма и перегрузки операторов в Python.

## Требования

- Python 3.6+

## Установка

Не требует установки дополнительных пакетов.

## Использование

```bash
python modify.py
```

## API

### Класс `My_list`

Наследует от встроенного класса `list`.

#### Инициализация

```python
l1 = My_list(1, 2, 3)  # Создает список [1, 2, 3]
l2 = My_list(3, 4, 5)  # Создает список [3, 4, 5]
```

#### Операции

##### Вычитание `-`

**1. Вычитание списка из списка** — возвращает элементы первого списка, которые отсутствуют во втором:

```python
l3 = l1 - l2  # My_list(1, 2) — элементы, которых нет в l2
l4 = l2 - l1  # My_list(4, 5) — элементы, которых нет в l1
```

**2. Вычитание скаляра из списка** — вычитает число из каждого элемента:

```python
l5 = l1 - 1  # My_list(0, 1, 2) — каждый элемент уменьшен на 1
```

##### Деление `/`

Делит каждый элемент списка на делитель:

```python
l6 = l1 / 5  # My_list(0.2, 0.4, 0.6) — каждый элемент разделен на 5
```

**Исключение:** Деление на ноль вызывает ошибку `ValueError`:

```python
l1 / 0  # ValueError: Division by zero is not allowed.
```

#### Строковое представление

```python
print(l1)  # My_list contains: (1, 2, 3)
```

## Примеры

```python
l1 = My_list(1, 2, 3)
l2 = My_list(3, 4, 5)

# Вычитание списков
print(l1 - l2)  # My_list contains: (1, 2)
print(l2 - l1)  # My_list contains: (4, 5)

# Вычитание скаляра
print(l1 - 1)  # My_list contains: (0, 1, 2)

# Деление на скаляр
print(l1 / 5)  # My_list contains: (0.2, 0.4, 0.6)

# Комбинированные операции
result = (l1 - 1) / 2  # (1-1)/2=0, (2-1)/2=0.5, (3-1)/2=1
print(result)  # My_list contains: (0.0, 0.5, 1.0)
```

## Внутренняя реализация

### Метод `__sub__`

Перегружает оператор вычитания:
- Если второй операнд — это `My_list`, возвращает элементы, которые не в нём
- Иначе вычитает скаляр из каждого элемента

### Метод `__truediv__`

Перегружает оператор деления `/`:
- Проверяет, что делитель не равен нулю
- Делит каждый элемент на делитель

### Метод `__str__`

Переопределяет строковое представление для информативного вывода.
