## 🎮 Проекты

### 1. 🐾 Animal - Tamagotchi

**Описание:** Виртуальная игра-симуляция по уходу за цифровым питомцем.

**Технологии:** Python, многопоточность (threading)

**Возможности:**
- Питание питомца
- Развлечение питомца
- Мониторинг сатости и счастья
- Асинхронный жизненный цикл

**Запуск:**
```bash
cd Animal
python animal.py
```

**Статус:** ✅ Работает

---

### 2. 📝 Modify - Custom List

**Описание:** Кастомный класс списка с перегрузкой операторов.

**Технологии:** Python OOP, операторы

**Возможности:**
- Вычитание списков (разность множеств)
- Вычитание скаляра из элементов
- Деление элементов на скаляр
- Кастомное строковое представление

**Запуск:**
```bash
cd Modify
python modify.py
```

**Пример:**
```python
l1 = My_list(1, 2, 3)
l2 = My_list(3, 4, 5)
print(l1 - l2)   # My_list contains: [1, 2]
print(l1 / 2)    # My_list contains: [0.5, 1.0, 1.5]
```

**Статус:** ✅ Работает

---

### 3. 🌍 Sun_system - Solar System Simulator

**Описание:** Интерактивная визуализация Солнечной системы с движением планет.

**Технологии:** Pygame, астрономия/математика

**Возможности:**
- Визуализация 8 планет
- Орбитальные движения
- Интерактивный наведение мышкой для информации о планетах
- Счетчик дней (на основе земного года)

**Требования:**
- Python 3.6+
- pygame
- PNG изображения планет

**Запуск:**
```bash
cd Sun_system
python body.py
```

**Статус:** ✅ Синтаксис верный (требует Pygame и изображений)

---

### 4. 📐 Triangulate - Triangle Analyzer

**Описание:** Анализ коллекции точек для поиска треугольников с минимальной и максимальной площадью.

**Технологии:** Python, геометрия (формула Герона)

**Возможности:**
- Чтение точек из файла
- Генерация всех возможных треугольников
- Вычисление площади по формуле Герона
- Поиск экстремальных треугольников

**Запуск:**
```bash
cd Triangulate/triangules
python tri.py
```

**Входные данные:** `plist.txt` с точками в формате `[x,y][x,y]...`

**Статус:** ✅ Работает отлично

**Описание:** Классическая игра Крестики-Нолики с графическим интерфейсом.

**Технологии:** Pygame, GUI

- Графический интерфейс 3x3

**Требования:**

- pygame

**Запуск:**
```bash

```

**Статус:** ✅ Синтаксис верный (требует Pygame)
