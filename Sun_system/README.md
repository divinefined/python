# Solar System Simulator

Интерактивная визуализация Солнечной системы с движением планет вокруг Солнца в реальном времени.

## Описание

Программа создает визуальную симуляцию нашей Солнечной системы с:
- **Солнцем** в центре экрана
- **8 планетами** с корректными орбитальными периодами
- **Астрономически точными размерами** (в услов. единицах)
- **Интерактивностью** — наведите мышь на планету, чтобы увидеть информацию

## Требования

- Python 3.6+
- **pygame** — библиотека для графики
- Файлы изображений планет в папке проекта

### Система координат

- **Центр**: центр экрана (WIDTH/2, HEIGHT/2)
- **Расстояние**: линейное масштабирование от реальных расстояний
- **Угол**: вращение по часовой стрелке от северной точки
- **Счетчик дней**: основан на орбите Земли

## Архитектура

### Класс `Planet`

Представляет планету в системе.

**Параметры:**
- `name` — имя планеты
- `radius` — видимый радиус на экране
- `distance` — расстояние орбиты от центра (Солнца)
- `period` — орбитальный период в днях (масштабируется на 0.05 для видимости)
- `image_path` — путь к файлу изображения планеты

**Методы:**
- `update(dt)` — обновление положения планеты на основе времени
- `draw(screen)` — отрисовка планеты на экран
- `is_mouse_over()` — проверка, находится ли мышь над планетой
- `get_info()` — получить информацию о планете

## Эффекты и вычисления

### Движение планет

Положение планеты определяется:
```
x = cos(угол) * расстояние + центр_x
y = sin(угол) * расстояние + центр_y
угол = угол + (dt / период) * 360°
```

Где **dt** — время в миллисекундах с последнего кадра.

### Масштабирование времени

- Орбитальные периоды масштабируются на **0.05** для видимого движения
- FPS = 600 для плавности (можно изменить)
