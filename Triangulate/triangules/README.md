# Triangle Analyzer

Программа для анализа треугольников, сформированных из набора точек. Находит треугольники с минимальной и максимальной площадью.

## Описание

Программа читает координаты точек из текстового файла, генерирует все возможные комбинации треугольников и определяет:
- Треугольник с **минимальной площадью**
- Треугольник с **максимальной площадью**

Площадь треугольника вычисляется по формуле Герона.

## Требования

- Python 3.6+
- Стандартная библиотека Python (math, re, os)

## Установка

Не требует установки дополнительных пакетов. Просто убедитесь, что у вас установлен Python.

## Использование

### Формат входного файла

Файл `plist.txt` должен содержать координаты точек в формате:
```
[x1,y1][x2,y2][x3,y3]...[xn,yn]
```

Пример:
```
[0.0,0.0][5.0,48.0][50.0,8.0][25.0,6.0][15.0,16.0][29.0,2.0]
```

### Запуск программы

```bash
python tri.py
```

Программа автоматически найдет файл `plist.txt` в той же папке и выведет результаты.

### Пример вывода

```
Найдено точек: 6
Точек после преобразования: 6
Треугольник с минимальной площадью: Triangle(Point(25.0, 6.0), Point(15.0, 16.0), Point(29.0, 2.0)) (площадь: 2.372174397451135e-06)
Треугольник с максимальной площадью: Triangle(Point(0.0, 0.0), Point(5.0, 48.0), Point(50.0, 8.0)) (площадь: 1180.0)
```

## Классы и функции

### Класс `Point`
Представляет точку в двумерном пространстве.

**Методы:**
- `__init__(x, y)` - Инициализация точки с координатами
- `__repr__()` - Строковое представление точки
- `distance_to(other)` - Вычисление расстояния до другой точки

### Класс `Triangle`
Представляет треугольник, образованный тремя точками.

**Методы:**
- `__init__(p1, p2, p3)` - Инициализация треугольника
- `__repr__()` - Строковое представление треугольника
- `area()` - Вычисление площади по формуле Герона
- Операторы сравнения (`<`, `<=`, `==`, `!=`, `>`, `>=`) для сравнения по площади

### Функции

- `read_points_from_file(file_path)` - Чтение точек из файла
- `find_min_max_triangles(points)` - Поиск треугольников с минимальной и максимальной площадью

## Алгоритм

1. Чтение координат точек из файла с помощью регулярного выражения
2. Генерация всех возможных комбинаций треугольников из набора точек (C(n,3))
3. Вычисление площади каждого неырожденного треугольника
4. Нахождение треугольников с минимальной и максимальной площадью

**Сложность:** O(n³) по времени, где n - количество точек

## Особенности

- ✅ Автоматическое определение пути к файлу независимо от рабочей директории
- ✅ Проверка на вырожденные треугольники (с нулевой площадью)
- ✅ Обработка ошибок при парсинге координат
- ✅ Подробный отладочный вывод

## Примечания

- Программа игнорирует треугольники с нулевой площадью (вырожденные)
- Координаты могут быть целыми числами или дробными (float)
- Поддерживаются отрицательные координаты
